📺 YouTube Clone (Home Page) 🎬
A simple replica of the YouTube home page built using HTML, CSS, and Bootstrap. This project focuses on layout and design, mimicking the familiar interface of YouTube.

✨ Features
✅ Responsive design 📱
✅ Video thumbnails 🎞️
✅ Sidebar navigation 📂
✅ Search bar 🔍
